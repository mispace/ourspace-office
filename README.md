# ourspace-office

Página estática do "Escritório Living OS" (renderizador em pixel art).

Este repositório contém **apenas o desenho** e sprites de licença livre. Os dados
(quem está fazendo o quê) são publicados **criptografados** (`agents.enc` / `status.enc`,
AES-256-GCM) por um processo à parte, a partir de um repositório privado — e só são
legíveis com a senha, digitada no navegador. Sem a senha, esta página não mostra
nenhum dado.

Sprites: Modern Interiors (versão free) por LimeZu — https://limezu.itch.io/moderninteriors
(licença não-comercial; uso interno).
